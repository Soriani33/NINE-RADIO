# NINE RADIO

Radio locale intelligente, stable et fluide.

## Installation
Ouvrez simplement `index.html` dans un navigateur moderne.

## Fonctionnalités
- Stations locales, Google Drive, Spotify et SoundCloud.
- Mode Admin (mot de passe par défaut : `radio`).
- Persistance des données via IndexedDB et localStorage.
- Visuels intelligents basés sur les titres en cours.
- Tickers d'actualités, économie et sport.
- Cartes d'infos : Météo, Lune, Marchés, Astro.

## Utilisation
1. Cliquez sur "Mode Admin" en haut à droite.
2. Entrez le mot de passe `radio`.
3. Ajoutez des stations ou utilisez le "Mix Rapide".
4. Exportez votre configuration en JSON pour la sauvegarder ailleurs.
